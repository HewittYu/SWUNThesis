# 西南民族大学本科毕业设计论文模版
本项目给出了一个毕业设计论文的模板，可以忽略一些LaTeX的语法细节进行高效论文编写。

使用过程中若遇到任何问题，欢迎联系我：**[hi[at]changkun.us](mailto:hi@changkun.us)**。

# 注意事项
- 适用于西南民族大学学生
- 目前仅支持本科生毕业论文
